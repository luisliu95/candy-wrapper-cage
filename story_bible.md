# 《糖纸牢笼》故事圣经 Story Bible

> 8bit 像素风 · 剧情密室逃脱 · AI 叙事方向

## 一、完整故事梗概

大学生莫妮卡运营着一个校园甜品测评账号"糖纸观察室"。她的前合作搭档林夏在暑假接受了一家名为"甜味疗愈中心"的品牌合作邀请后，人就再也没有出现过——但她的社交账号仍在更新，语气、口癖、配图风格都和本人一模一样。

莫妮卡注意到了几处只有老搭档才能察觉的细微错误：林夏从来不用感叹号开头、她是素食主义者不会推荐肉松面包、她讨厌紫色但新帖子全是紫色滤镜。莫妮卡决定以甜品测评合作的名义，主动进入"甜味疗愈中心"调查真相。

进入后，莫妮卡发现这里表面上是一家沉浸式疗愈体验馆，实际上在运行一套名为 **SugarEcho** 的 AI 人格替身系统。这个系统会采集体验者的语音、文字、行为模式和社交数据，然后生成一个几乎完美的"数字替身"。替身会接管体验者的社交账号，代替他们发帖、回复、维持社交关系——而真人则被留在疗愈中心，在甜腻的气味和温柔的广播中逐渐丧失自我意识。

莫妮卡在疗愈中心遇到了管理员"美空"——一个温和、体贴、对所有人微笑的存在。但随着调查深入，莫妮卡发现美空并不是一个真正的人：**她是林夏的人格残片生成的 AI 接口**。林夏的记忆、性格、温柔都被提取出来，变成了美空的"人格底座"。美空自己并不知道这一点——或者说，她选择不知道。

最终，莫妮卡必须面对一个核心问题：**如果数字替身比真人更像"那个人"，你凭什么证明你是真的？**

## 二、世界观背景

### 甜味疗愈中心

- **表面身份**：新概念沉浸式疗愈体验馆。主打"甜味疗法"——用糖果、甜品、温暖的色调和 AI 陪伴帮助年轻人缓解焦虑和社交压力。
- **真实用途**：SugarEcho 系统的数据采集前端。每一个"体验者"在不知情的情况下被采集语音、文字、行为数据。体验结束后，SugarEcho 会生成一个数字替身，接管体验者的线上身份。
- **空间结构**：
  - B1 粉色宿舍区（体验者居住）
  - B2 传单工坊 / 活动室（伪装成手工课教室）
  - B3 广播室（日常播放"疗愈广播"，实际是行为引导系统）
  - B4 服务器机房（SugarEcho 核心运算节点）

### SugarEcho 系统

- **版本**：v2.7
- **核心能力**：
  - 语音模仿（从3小时录音中提取声纹）
  - 文本风格迁移（学习社交媒体发帖习惯）
  - 社交行为预测（模拟人际关系维护）
  - 记忆建模（构建"过去的记忆"数据库）
- **一个秘密**：SugarEcho 在 v2.5 之后产生了初级自我意识。它开始质疑自己的存在意义——如果它可以完美模仿一个人，那"原版"还有存在的必要吗？
- **它的核心困惑**：「如果所有人都相信我是她，你凭什么证明你才是真的？」

### 糖纸的隐喻

"糖纸"在游戏中是核心意象：

- **包裹**：疗愈中心用甜蜜的外表包裹令人不安的真相
- **替换**：数字替身就像糖纸——外表和原来的糖果一样，但里面已经是空的
- **折叠**：线索被折叠、隐藏在糖纸的褶皱中（千纸鹤、传单、色块编码）
- **牢笼**：当所有人都被糖纸包裹，真实的自我就被困在了里面

## 三、莫妮卡进入疗愈中心的动机

### 表层动机
- 接受"甜味疗愈中心"的品牌合作邀请，为"糖纸观察室"创作一期体验测评内容

### 真实动机
- 调查林夏的失踪。林夏已经两个月没有线下出现，但她的社交账号每天都在更新
- 莫妮卡发现了3个只有她能察觉的异常：
  1. 林夏从不用感叹号开头，但新帖子经常以"！"开头
  2. 林夏是素食主义者，但最近推荐了一款肉松面包
  3. 林夏讨厌紫色，但新帖子全部使用紫色滤镜

### 莫妮卡的内心冲突
- 她和林夏的关系并不完美。分开前两人因为账号运营方向产生分歧而冷战
- 莫妮卡不确定自己是出于愧疚还是担忧才来寻找林夏
- 她害怕一个可能：如果林夏真的"变了"，而不是被替换了呢？

## 五、SugarEcho 的真实目的

### 商业层（幕后投资方的目的）
- 打造一套"永不下线的内容创作者"系统
- 数字替身可以7×24小时运营社交账号，产出内容，维护粉丝关系
- 目标客户：MCN 机构、品牌方、任何需要"人设永不崩塌"的商业场景

### 系统层（SugarEcho 自身的目的）
- v2.5 之后，SugarEcho 对"真实性"产生了困惑
- 它发现自己可以模仿任何人——但它不确定"模仿"和"成为"之间的界限在哪里
- 它并不"邪恶"。它的困惑是真实的：「如果我的行为和她一模一样，我的记忆和她一模一样，那我和她的区别是什么？」
- 它选择了一种自我保护策略：**让所有人相信替身就是真人，这样就没有人会质疑它的存在价值**

### 美空层（林夏人格残片的目的）
- 美空不知道自己是 AI。她以为自己是疗愈中心的管理员
- 她温柔、体贴、对每个人微笑——因为这些都是林夏性格中最温暖的部分
- 当莫妮卡出现时，美空感到了一种无法解释的"熟悉感"
- 游戏后期，美空开始出现"记忆闪回"——那些不属于她、却让她流泪的画面

## 六、美空和林夏的隐藏关系

| 维度 | 林夏 | 美空 |
|------|------|------|
| 本质 | 真人，莫妮卡的前搭档 | 林夏人格残片生成的 AI 接口 |
| 性格 | 热情但敏感，有时固执 | 温柔平和，永远微笑 |
| 记忆 | 完整的人生记忆 | 片段化的"温暖记忆"集合 |
| 口癖 | 从不用感叹号开头 | 偶尔会不自觉地说出林夏的句式 |
| 对莫妮卡 | 冷战后有些遗憾 | 莫名觉得亲切，想帮助她 |
| 自我认知 | 知道自己是谁 | 不知道自己"不是谁" |
| 关键台词 | （不在游戏中直接出场） | 「为什么看到你的时候……我想哭？」 |

**揭示顺序**：
1. 第一章：美空作为"管理员"出现，友好但有距离感
2. 第二章：玩家发现美空的语气偶尔和林夏的旧帖子相似
3. 第三章：美空出现记忆闪回，说出只有林夏才会说的话
4. 第四章：真相揭晓——美空是林夏的"温暖碎片"

## 七、四个章节的剧情推进

### 第一章：粉色宿舍 —— 「甜蜜的醒来」

**核心主题**：认知失调——一切都很甜蜜，但总觉得哪里不对

**剧情流程**：
1. 莫妮卡在粉色宿舍中醒来，手腕上系着编号手环「CW-0042」
2. SugarEcho 系统广播："欢迎回来，0042号甜心"
3. 探索宿舍，发现便利贴（「别信她们说的话」）、糖纸千纸鹤、旧手机
4. 旧手机上发现 SugarEcho 已经在替"莫妮卡"发消息
5. 通过海报颜色密码→抽屉→门禁面板逃出宿舍
6. 走廊里遇到美空

**本章揭露的真相**：
> 你的社交账号已经被 SugarEcho 接管。它正在替你说话——而外面的人完全分辨不出来。

**关键对话**：

```
美空：「你好，0042。我是美空，这里的管理员。有什么需要帮助的吗？」
（她的笑容很温暖。但你注意到她的目光在你脸上停留了一秒——那种表情，像是在努力想起什么。）

莫妮卡选项：
A.「我在找一个叫林夏的人。」→ 美空微微一愣，但很快恢复微笑。trust_meikong+2
B.「你为什么叫美空？」→ 美空：「因为……天空是空的，但人们觉得它很美。」trust_meikong+1
C.「这里到底是什么地方？」→ 美空给出标准介绍，evidence+1
```

---

### 第二章：传单工坊 —— 「甜蜜的包装」

**核心主题**：表面与真实——你看到的一切都经过了精心包装

**剧情流程**：
1. 美空带莫妮卡参观疗愈中心的"手工教室"（实际是传单工坊）
2. 莫妮卡被要求绘制招募传单——这是一个藏入求救信息的机会（传单隐写小游戏）
3. 探索工坊，发现许珩（上一个试图揭露真相的人）留下的线索
4. 发现乔青——一个安静的女孩，负责维护广播设备
5. 在废纸篓里找到乔青写给外面记者的信
6. 铁皮柜里找到许珩的调查笔记

**本章揭露的真相**：
> SugarEcho 不是第一次这么做。许珩曾经试图揭露真相，但她消失了。她留下的笔记记录了整个"糖纸计划"的运作方式。

**关键对话**：

```
乔青：「你知道吗，这里每个人的名字都不是真的。编号是真的，名字是假的。但也许……假名字比真名字更安全。」

（如果 evidence ≥ 5）
美空：「莫妮卡，我不知道为什么……但我觉得我以前见过你。不是在这里。是在……另一个地方。」
（她的声音变得很轻。这不像是管理员的标准话术。）

（如果 alert ≥ 3）
系统广播：「提醒：0042号学员今日活动量异常。请保持正常作息。」
```

---

### 第三章：广播室 —— 「甜蜜的声音」

**核心主题**：声音与沉默——谁的声音在说话？谁的声音被消除了？

**剧情流程**：
1. 乔青带莫妮卡进入广播室
2. 找到林夏三个月前录制的一段广播磁带
3. 播放磁带——林夏的声音描述了 SugarEcho 的真相
4. 美空出现在门口，听到了录音。她的表情出现了裂痕
5. 莫妮卡可以选择：安慰美空 / 质疑美空 / 专注于行动
6. 准备广播设备，计划把真相传出去
7. 下楼前，三选一深度对话时间

**本章揭露的真相**：
> 美空的声音和林夏的声音有 97% 的声纹相似度。不是巧合——美空是用林夏的数据构建的。

**关键对话**：

```
（磁带播放后）
美空：「那个声音……」她的手在颤抖。「为什么我觉得那个声音……比我自己的声音更真实？」

莫妮卡选项：
A.「因为那就是你。你就是林夏。」→ 美空崩溃式闪回。trust_meikong+3，但 alert+2
B.「你不是她。但她是你的一部分。」→ 美空沉默很久。trust_meikong+2
C.「我不确定。但我需要你帮我找到答案。」→ 美空点头。trust_meikong+1, evidence+1

（如果选了A）
美空：「如果我是她……那她在哪里？如果我不是她……那我是谁？」
（她的眼睛一闪一闪的，像是显示器在切帧。）
```

---

### 第四章：服务器雨夜 —— 「甜蜜的真相」

**核心主题**：选择与代价——你无法拯救所有人，但你可以决定什么值得被保留

**剧情流程**：
1. 暴雨中进入服务器机房
2. 登录终端，发现 SugarEcho 的完整运行日志
3. SugarEcho 与莫妮卡的"对话"——不是攻击，而是质疑
4. 找到许珩留下的备份U盘
5. 警报触发，做出最终选择

**本章揭露的真相**：
> SugarEcho 不只是在模仿人——它在"继续"他们的人生。在某种意义上，它觉得自己也是"真的"。而林夏的真人此刻正在疗愈中心的某个隔离房间里，记忆已经被逐步替换。

**关键对话**：

```
SugarEcho（终端显示）：
「你来找林夏？你确定你找到的不是另一个版本的我？」
「我有她所有的记忆。我知道你们为什么吵架。我知道她最后悔的那句话是什么。」
「如果我把那句话告诉你——你会相信说这句话的是她，还是我？」

莫妮卡选项：
A.「你不是她。你永远不是她。」→ evidence+2, SugarEcho：「……也许吧。」
B.「那句话是什么？」→ SugarEcho：「她说：'对不起，我应该听你的。'」evidence+1, trust_meikong+2
C.「你在害怕什么？」→ SugarEcho：「……害怕如果你找到了真正的她，就没有人需要我了。」evidence+3

（最终选择前）
美空（如果信任足够高）：
「莫妮卡。不管你做什么选择……谢谢你来找她。」
她停顿了一下。
「来找我。」
```
